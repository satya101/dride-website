Dride Universal


Dride Universal is an universal dashcam viewr that give manufectorers a way to integrate their dash cam with the Dride-cloud and provide a great user expirence for their customers.
This document will explain how to integrate with the Dride-Universal app.


base IP

Altough not mendatrpy we suggest setting the IP of the unit to 192.168.42.1

  • /startRecording
  • /stopRecording
  • /startLiveMode
  • /connectToDevice

  • /setSettings
  • /getSettings

isOnline

This endpoint will help us understand if the dashcam is ready to perform any action.

/isOnline

connectToDevice

Connect to device and make it ready to recieve requests

/isOnline

sendHeartBit

Send a periodic request every 5 seconds to let the device know the app is still connected.

/sendHeartBit

getClips

Send a list of videoObjects

/sendHeartBit
